# 📊 Анализ команд LLMevangelist бота

## 🔴 Критические дубли (90% пересечения)

### `/optimize` + `/recommend`
- **Что делает**: оба анализируют проекты и дают рекомендации
- **Решение**: объединить в `/optimize` с флагами: `--brief`, `--detailed`

### `/top` + часть `/insights`
- **Что делает**: оба показывают топ моделей
- **Решение**: `/insights` показывает дайджест, `/top` — только топ

## 🟡 Сложные команды UX

| Команда | Проблема | Решение |
|---------|---------|--------|
| `/models` | 371 модель в выводе (2000+ токенов) | Добавить фильтры: `free`, `paid`, `--limit 10` |
| `/stats` | Выводит ID, каналы | Убрать, оставить только usage |
| `/scan` | Длинные пути | Сократить: `projects/xxx` вместо `/root/LabDoctorM/projects/xxx` |

## 🟢 Приоритет действий

1. **Немедленно**: объединить `/optimize` + `/recommend`
2. **В течение недели**: рефакторинг `/models`, `/stats`, `/scan`  
3. **В течение месяца**: оптимизация `/insights`
