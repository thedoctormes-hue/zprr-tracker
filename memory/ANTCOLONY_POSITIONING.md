## 👷 Архитектурные паттерны (работодателям)

1. **Planner → Coder → Verifier** — sequential handoff для сложных задач
2. **Redis Message Bus** — агент-агент коммуникация
3. **Capability Registry** — динамический выбор агентов по навыкам
4. **Feedback Loop** — обучение на успешных паттернах
