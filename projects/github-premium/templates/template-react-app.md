## 🚀 Quick Start
### Требования
- Node.js 18+
- npm 9+

### Установка
```bash
git clone https://github.com/{{OWNER}}/{{REPO}}.git
cd {{REPO}}
npm install
```

### Настройка
Создайте `.env` файл:
```env
VITE_API_URL=https://api.shtab-ai.ru
```

### Запуск в режиме разработки
```bash
npm run dev
```

### Сборка для продакшена
```bash
npm run build
```
