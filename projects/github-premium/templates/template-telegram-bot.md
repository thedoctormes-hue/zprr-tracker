## 🚀 Quick Start
### Требования
- Python 3.10+
- aiogram 3.x
- Telegram Bot Token

### Установка
```bash
git clone https://github.com/{{OWNER}}/{{REPO}}.git
cd {{REPO}}
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Настройка
Создайте `.env` файл:
```env
BOT_TOKEN=your_telegram_bot_token
```

### Запуск
```bash
python main.py
```
