# Методология диагностики ноутбука

## Этап 1: Подключение
1. Настроить SSH сервер на ноутбуке
2. Создать reverse SSH туннель
3. Проверить соединение

## Этап 2: Сбор метрик
1. CPU: `lscpu`, `htop`, `uptime`
2. RAM: `free -h`, `vmstat`
3. Disk: `df -h`, `du -sh /*`, `smartctl`
4. Temp: `sensors`, `hddtemp`

## Этап 3: Оптимизация
1. Swappiness → 10
2. TRIM SSD
3. Отключить сервисы
4. TLP + thermald
