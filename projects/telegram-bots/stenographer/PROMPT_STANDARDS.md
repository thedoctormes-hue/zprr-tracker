# Стандарты промптов Stenographer

## Формат блоков (КРИТИЧНО для LLM)

```
[КОНТЕКСТ]
Кто ты, какая роль, цель

[ПРАВИЛА]
Что делать, как делать, ограничения

[ДАННЫЕ]
{{ПЕРЕМЕННАЯ_С_ДАННЫМИ}}

[ВЫХОД]
Что вернуть
```

## Важность порядка

LLM видит контекст последовательно. Нарушение порядка блоков приводит к:
- 50% падение качества на валидации
- Неправильная интерпретация инструкций
- Потеря данных при длинных контекстах

## Примеры

### Правильно ✅
```
[КОНТЕКСТ]
Ты — стенографист.

[ПРАВИЛА]
...

[ДАННЫЕ]
{{ТЕКСТ_ДЛЯ_АНАЛИЗА}}

[ВЫХОД]
...
```

### Неправильно ❌
```
Ты — стенографист. Убери слова-паразиты...

Текст:
---
{{ТЕКСТ_ДЛЯ_АНАЛИЗА}}
```

## Плейсхолдеры

Поддерживаются любые `{{ПЕРЕМЕННАЯ}}`. Код автоматически подставит данные.
