# stenographerobot

## Описание
Telegram-бот для транскрибации аудиозаписей на русском языке с использованием LLM (OpenRouter API). Принимает голосовые сообщения, аудиофайлы и документы, преобразует их в текст, очищает от слов-паразитов, а также умеет выделять задачи и рефлексию из текста.

## Архитектура

```
stenographerobot/
├── main.py              # Точка входа, запуск бота, проверка PID
├── requirements.txt    # Зависимости
├── .env                 # Токены и ключи
├── bot/
│   ├── handlers.py      # Обработчики команд и сообщений
│   └── keyboards.py    # Inline-клавиатуры
├── core/
│   ├── config.py        # Конфигурация, загрузка .env
│   └── api.py           # Интеграция с OpenRouter, ffmpeg
├── prompts/
│   ├── clean.txt        # Промпт для очистки текста
│   └── protocol.txt     # Промпт для протокола + задач + рефлексия
└── utils/
    └── files.py         # Проверка файлов и размера
```

**Поток данных (магия):**
1. Пользователь отправляет аудио → `handlers.py`
2. Файл скачивается, конвертируется в WAV (ffmpeg)
3. `api.py` отправляет в OpenRouter (Gemini 2.5 Flash Lite)
4. LLM возвращает **сырой текст**
5. LLM очищает от паразитов
6. LLM генерирует **ВЫВОДЫ** (протокол + задачи + рефлексия)
7. Бот отправляет **2 файла**: "Чистый текст" + "Выводы"

## Что работает

- Приём голосовых, аудио и документов (до 50 МБ)
- Транскрибация через OpenRouter API (Gemini 2.5 Flash Lite)
- Очистка текста от слов-паразитов, повторов, междометий
- **Магия:** мгновенные выводы (протокол + задачи + рефлексия)
- Защита от двойного запуска (PID-файл)
- Удаление вебхука при старте
- Повторные попытки при ошибках API
