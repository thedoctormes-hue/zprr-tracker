# TeamViewer Host для Xiaomi Stick (АВТОМАТИЧЕСКИЙ ДОСТУП)

## Разница QuickSupport vs Host:
- **QuickSupport** - временный ID, требует подтверждения
- **Host** - постоянный ID, автозапуск, unattended access

## Установка:

### Вариант 1: Из Google Play (проще)
1. Google Play → "TeamViewer Host"
2. Установи приложение
3. Запусти - получишь постоянный ID

### Вариант 2: APK напрямую
```bash
# С сайта TeamViewer (работает в РФ)
wget https://download.teamviewer.com/download/file/TeamViewerHost.apk
# Установи через файл-менеджер
```

## Настройка автозапуска:
1. Settings → Apps → TeamViewer Host → Battery → "No restrictions"
2. Settings → Device Preferences → Startup applications → добавить Host
