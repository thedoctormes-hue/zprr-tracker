## 📝 План задач

### 1. Авто-деплой (25 новых серверов)
- **Скрипт**: `deploy_xray_server.sh` — копирует оптимизированный конфиг, применяет sysctl, настраивает systemd.
- **Генерация ключей**: уникальные Reality private keys для каждого сервера.
- **Сервера**:
  - 20 иностранных: SNI `google.com`/`cloudflare.com`
  - 5 российских: SNI `yandex.ru`/`mail.ru` (обход DPI РФ)

### 2. Оптимизация Xray под 60k
- `policy.levels.0`: `connIdle=300`, `handshake=4`, `floodAttack=reject`
- `inbounds[0].allocate.concurrency`: увеличить до 5000+
- Включить `stats` + `api` для мониторинга нагрузки.
- `sockopt`: `tcpKeepAliveIdle=30`, `tcpNoDelay=true`

### 3. Бот: Мульти-серверность
- Расширить выбор серверов (пока только WARSAW/FLORIDA).
- Авто-балансировка: новый клиент → сервер с меньшей нагрузкой.
- Кнопки выбора геолокации (Европа/США/РФ).

### 4. Бот: AI-мозги (интеграция LLM)
- Команда `/ai_diagnose` — анализ логов, рекомендации.
- Использовать OpenRouter бесплатные модели:
  - `google/gemma-4-31b-it:free` (контекст 262k)
  - `qwen/qwen3-next-80b-a3b-instruct:free` (контекст 256k)
- Чтение GitHub Xray docs через `web_fetch`.

### 5. Мониторинг (27 серверов)
- Health-check каждые 5 мин (пинг, порты 443/10086, статус сервисов).
- Алерты в ТГ при падении (через бота).
- Xray stats парсинг (онлайн клиенты).

### 6. Российские серверы (спец-настройки)
- SNI: `yandex.ru`, `mail.ru` (менее заметно для РКН).
- Порты: 443, 2053, 2083 (менее сканируются).
- Тестирование обхода DPI местных провайдеров.

### 7. Subscription сервер
- Объединение конфигов со всех 27 серверов в единый файл.
- Эндпоинт: `https://185.138.90.150:9000/subscription`
- Поддержка мульти-сервера в ссылках.

### 8. Тестирование
- Нагрузочное тестирование (имитация 60k подключений).
- Проверка стабильности на реальных клиентах.
- Failover тестирование (отказ одного сервера).

## ⚡ Риски и решения
- ⚠️ Блокировка IP провайдерами → смена портов, SNI
- ⚠️ Перегрузка серверов → балансировка, `ulimit` 524288+
- ⚠️ Сложность управления 27 серверами → авто-деплой, централизованный бот
